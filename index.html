Projet RH Niveau de satisfaction Collaborateurs

Je partage projet réalisé en un délai court (48h). Le but du projet est de proposer un dashboard pour un cabinet de conseil souhaitant aider ses clients à améliorer le bien-être et la performance de leurs employés.

Lien vers le dataset : Employee Productivity and Satisfaction HR Data

Étapes du projet :

  1. Étape 1 : Documentation sur les indicateurs clés de la satisfaction et de la productivité des employés (KPI).
    • Informations générales des employés : salaire moyen, âge moyen, temps moyen d'embauche.
    • Satisfaction : Note moyenne, catégories (Détracteurs, Passifs, Promoteurs), calcul du ENPS.
    • Productivité : Productivité moyenne, projets terminés, score moyen de feedback.
  2. Étape 2 : Analyse exploratoire des données (EDA).

    L’EDA permet de détecter les erreurs, identifier les valeurs aberrantes et trouver les relations intéressantes entre les variables.

    Voir le notebook : EDA Mission Data RH

  3. Étape 3 : Création du dashboard sur Power BI.

Projet Cyclistic - Entreprise de partage de vélos à Chicago

Il s’agit d’un projet réalisé dans le cadre de ma formation de Data Analyst. L'objectif du projet est de proposer un dashboard pour une entreprise de partage de vélos qui souhaite augmenter le nombre de ses abonnés.

Étapes du projet :

  1. Étape 1 : Documentation sur les indicateurs clés dans le domaine du transport, et plus particulièrement des vélos. Identification des différents leviers pour l’augmentation des ventes d’abonnements de transport.
    • Informations générales sur l’utilisation des vélos
    • Analyse du comportement de la clientèle
  2. Étape 2 : Analyse exploratoire des données (EDA).
  3. Étape 3 : Création du dashboard sur Power BI.
